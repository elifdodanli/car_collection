# Car Collection 🏎️✨

A sleek, aesthetic mobile application for car enthusiasts to curate and showcase their personal automotive collection. Built with a focus on clean design and seamless user experience, Car Collection allows users to track their dream cars, save favorites, and organize their garage with style.

## 🚀 Features

### Implemented
* **Authentication:** Secure user registration and login flow powered by Firebase Auth.
* **Curated Dashboard:** Aesthetic, grid-based layout to view your car collection.
* **Dynamic Data:** Real-time synchronization with Cloud Firestore to manage your garage.
* **Onboarding Flow:** Engaging walkthrough for new users to personalize their profile.

### Upcoming
* **Favorites System:** Toggle and filter your most-loved vehicles.
* **User Profile:** Manage personal details and collection statistics.
* **Cloud Storage:** Direct image upload from the gallery to Firebase Storage.
* **OAuth Integration:** Google and Apple Sign-in support.

## 🛠 Tech Stack

* **Language:** Dart
* **Framework:** Flutter
* **Backend:** Firebase (Authentication, Cloud Firestore)
* **UI/UX:** Custom aesthetic theme with soft-coquette inspirations
